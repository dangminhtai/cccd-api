CCCD API Documentation

API để parse thông tin từ số CCCD (Căn cước công dân) 12 chữ số của Việt Nam. Lấy thông tin về mã tỉnh/thành phố, giới tính, năm sinh, thế kỷ và tuổi.

security

Secure API Keys

API key authentication với tier-based rate limiting.

link Base URL

Development: http://127.0.0.1:8000
Production: https://api.cccd-api.com

key Authentication

API sử dụng API key authentication qua header X-API-Key. Bạn có thể lấy API key tại Customer Portal.

Request Headers 
X-API-Key: your-api-key-here
Content-Type: application/json
info

Lấy API Key

  1. Đăng ký tài khoản tại Customer Portal
  2. Đăng nhập và tạo API key tại Keys Management
  3. Copy API key và sử dụng trong requests

POST /v1/cccd/parse
verified Official

Parse số CCCD 12 chữ số để lấy thông tin về mã tỉnh/thành phố, giới tính, năm sinh, thế kỷ và tuổi.

warning

Lưu ý: Chỉ chấp nhận POST method. OPTIONS trả về 405.

Body Parameters

cccd string(12)

Số CCCD 12 chữ số (pattern: ^[0-9]{12}$)

Required

province_version string

Phiên bản mapping: legacy_63 hoặc current_34 (mặc định: current_34)

Optional

curl -X POST http://127.0.0.1:8000/v1/cccd/parse \
  -H "Content-Type: application/json" \
  -H "X-API-Key: your-api-key-here" \
  -d '{
    "cccd": "079203012345",
    "province_version": "current_34"
  }'

warning Common Error Codes

Code Description
200 Request thành công
400 Validation error - CCCD format không hợp lệ hoặc thiếu field
401 API key không hợp lệ hoặc thiếu
405 Method không được hỗ trợ (chỉ POST được phép cho /v1/cccd/parse)
429 Vượt quá rate limit
500 Lỗi server

code Code Examples 

# Using requests library
import requests

url = "http://127.0.0.1:8000/v1/cccd/parse"
headers = {
    "X-API-Key": "your-api-key-here",
    "Content-Type": "application/json"
}
data = {
    "cccd": "079203012345"
}

response = requests.post(url, json=data, headers=headers)
print(response.json())

# Using Python SDK
from sdk.python.cccd_api import CCCDAPI

with CCCDAPI(api_key="your-api-key-here") as api:
    result = api.parse("079203012345")
    print(result.data['province_name'])  # Tp. Hồ Chí Minh
// Using fetch API
fetch('http://127.0.0.1:8000/v1/cccd/parse', {
  method: 'POST',
  headers: {
    'X-API-Key': 'your-api-key-here',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    cccd: '079203012345'
  })
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));
curl -X POST http://127.0.0.1:8000/v1/cccd/parse \
  -H "X-API-Key: your-api-key-here" \
  -H "Content-Type: application/json" \
  -d '{"cccd": "079203012345"}'
<?php
$url = 'http://127.0.0.1:8000/v1/cccd/parse';
$data = json_encode(['cccd' => '079203012345']);

$ch = curl_init($url);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_POSTFIELDS, $data);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'X-API-Key: your-api-key-here',
    'Content-Type: application/json'
]);

$response = curl_exec($ch);
curl_close($ch);
echo $response;
?>
