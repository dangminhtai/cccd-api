CCCD API Documentation

API để parse thông tin từ số CCCD (Căn cước công dân) 12 chữ số của Việt Nam

Overview Authentication Endpoints Rate Limits Error Codes Code Examples

Overview

CCCD API cung cấp endpoint để parse số CCCD 12 chữ số và lấy thông tin:

  • Mã tỉnh/thành phố (province_code)
  • Tên tỉnh/thành phố (province_name)
  • Giới tính (gender)
  • Năm sinh (birth_year)
  • Thế kỷ (century)
  • Tuổi (age)
Base URL:
Development: http://127.0.0.1:8000
Production: https://api.cccd-api.com

Authentication

API sử dụng API key authentication qua header X-API-Key.

Lấy API Key

  1. Đăng ký tài khoản tại Customer Portal
  2. Đăng nhập và tạo API key tại Keys Management
  3. Copy API key và sử dụng trong requests
X-API-Key: your-api-key-here

API Endpoints

GET /health

Kiểm tra trạng thái API server

Request

GET /health
Content-Type: application/json

Response (200 OK)

{
  "status": "ok"
}

POST /v1/cccd/parse

Parse số CCCD 12 chữ số để lấy thông tin

Lưu ý: Chỉ chấp nhận POST method. OPTIONS trả về 405.

Headers

Header Type Required Description
X-API-Key string Required API key để authenticate
Content-Type string Required Must be application/json

Request Body

Field Type Required Description
cccd string Required Số CCCD 12 chữ số (pattern: ^[0-9]{12}$)
province_version string Optional Phiên bản mapping tỉnh/thành phố: legacy_63 hoặc current_34 (mặc định: current_34)

Request Example

{
  "cccd": "079203012345",
  "province_version": "current_34"
}

Response (200 OK)

200 OK 
{
  "success": true,
  "data": {
    "province_code": "079",
    "province_name": "Tp. Hồ Chí Minh",
    "gender": "Nam",
    "birth_year": 2003,
    "century": 21,
    "age": 21
  },
  "is_valid_format": true,
  "is_plausible": true,
  "province_version": "current_34",
  "warnings": null
}

Response với Warnings

200 OK 
{
  "success": true,
  "data": {
    "province_code": "079",
    "province_name": null,
    "gender": "Nam",
    "birth_year": 2025,
    "century": 21,
    "age": null
  },
  "is_valid_format": true,
  "is_plausible": false,
  "province_version": "current_34",
  "warnings": [
    "province_code_not_found",
    "birth_year_in_future"
  ]
}

Error Responses

400 Bad Request

CCCD format không hợp lệ hoặc thiếu field

{
  "success": false,
  "is_valid_format": false,
  "data": null,
  "message": "CCCD không hợp lệ (cần là chuỗi số, độ dài 12)."
}
401 Unauthorized

API key không hợp lệ hoặc thiếu

{
  "success": false,
  "is_valid_format": false,
  "data": null,
  "message": "API key không hợp lệ hoặc thiếu."
}
429 Too Many Requests

Vượt quá rate limit

{
  "success": false,
  "is_valid_format": false,
  "data": null,
  "message": "Quá nhiều request. Giới hạn: 10 per 1 minute"
}

Rate Limits

Rate limits được áp dụng theo tier của API key:

Tier Requests per Minute Requests per Day
Free 10 1,000
Premium 100 Unlimited
Ultra 1,000 Unlimited
Rate Limit Headers:
Response headers chứa: X-RateLimit-Remaining, X-RateLimit-Reset, Retry-After

Error Codes

Status Code Description
200 Request thành công
400 Validation error - CCCD format không hợp lệ hoặc thiếu field
401 API key không hợp lệ hoặc thiếu
405 Method không được hỗ trợ (chỉ POST được phép cho /v1/cccd/parse)
429 Vượt quá rate limit
500 Lỗi server

Code Examples

Python (requests library)

import requests

url = "http://127.0.0.1:8000/v1/cccd/parse"
headers = {
    "X-API-Key": "your-api-key-here",
    "Content-Type": "application/json"
}
data = {
    "cccd": "079203012345"
}

response = requests.post(url, json=data, headers=headers)
print(response.json())

Using Python SDK

from sdk.python.cccd_api import CCCDAPI

with CCCDAPI(api_key="your-api-key-here") as api:
    result = api.parse("079203012345")
    print(result.data['province_name'])  # Tp. Hồ Chí Minh

JavaScript (fetch API)

fetch('http://127.0.0.1:8000/v1/cccd/parse', {
  method: 'POST',
  headers: {
    'X-API-Key': 'your-api-key-here',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    cccd: '079203012345'
  })
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));

cURL

curl -X POST http://127.0.0.1:8000/v1/cccd/parse \
  -H "X-API-Key: your-api-key-here" \
  -H "Content-Type: application/json" \
  -d '{"cccd": "079203012345"}'

PHP (cURL)

<?php
$url = 'http://127.0.0.1:8000/v1/cccd/parse';
$data = json_encode(['cccd' => '079203012345']);

$ch = curl_init($url);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_POSTFIELDS, $data);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'X-API-Key: your-api-key-here',
    'Content-Type: application/json'
]);

$response = curl_exec($ch);
curl_close($ch);
echo $response;
?>
